11. (2 Marks)
Fetch flats with more than 3 residents using HAVING
Select * from flats where flatid = (
Select flatid from residents group by flatid having count(*) > 3
);

Write a query to find the total maintenance amount collected for each flat and display
the FlatNumber along with total amount, sorted in descending order.

select flatNumber, amount from flats, Maintenance where flats.flatid = ANY
(Select flatid from Maintenance group by flatid) order by 2 desc;
