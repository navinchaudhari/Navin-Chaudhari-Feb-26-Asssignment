1) Create a view that shows all of the customers who have the highest ratings
=> Create view HightRateCust
	AS
   Select * from customers
   Where Rating = 
	(Select max(rating) from Customers);

2) Create a view that shows the number of salespeople in each city.
=> Create View SPeopleInCity
	AS
   Select city, count(*) from salespeople group by city;
		
		OR
	Create OR Replace View SPeopleInCity
		AS
	Select city, count(*) People from salespeople group by city;

3) Create a view that shows the average and total orders for each salesperson after
his or her name. Assume all names are unique
=> 	Create View AvgNTotalOrders
		AS
	Select Sname NAME, avg(AMT) "Average AMT", Sum(AMT) "Total AMT" from Orders, Salespeople
	Where Salespeople.Snum = Orders.Snum
	Group by Sname;

4) Create a view that shows each salesperson with multiple customers.
=> CREATE VIEW saleWithMultiCust
		AS
	SELECT Sname, Cname FROM Customers, Salespeople
	WHERE Customers.Snum = Salespeople.Snum
	AND Customers.Snum IN 
	(SELECT Snum FROM Customers
	GROUP BY Snum
	HAVING count(*) > 1);
