1. What is Java?
-> java is an High level Object oriented Programming Language use to develope
   various services and Applications, Developed by Sun Microsystem in 1995.

2. Is Java platform dependent or independent?
-> Java is Platfrom Independent but its JVM that is run time environment is
   is Platform Dependent.

3. What is JVM?
-> JVM stands for Java Virtual Machine. It is used to run java programs on different
   machines. It convert Byte code into machine code and thats why it is platform dependent.

4. What is JDK and JRE?
-> JDK is Acronym for JAVA Development Kit it is used to develope complete application.
   It has compiler, Debugger, library classes, JRE, JVM to run code.
   JRE is acronym for Java Runtime Environment is a software that provides necessary
   libraries and JVM to run Java Program.

5. Why is Java called “Write Once, Run Anywhere”?
-> In java code compile by compiler is converted into Byte code which is not machine specific
   code and after that Byte code is then converted into machine code by JVM which is platform 
   Dependent. So that any byte code of java can be run on any machine using that machines JVM.
